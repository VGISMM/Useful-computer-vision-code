# Program functionality
showAnnotations.cpp <annotation directory> <frame number> shows the frames along with annotations in the directory pointed to. The directory must contain frames named with numbers beginning with the number giving as second argument along with the annotation text file called: annotations.txt. Progressing forward in frames is controlled by “r” and backward by “w”. The program takes .png images

# Use
call build script “./build.sh”

“chmod +x build.sh” if necessary

execute “./showAnnotations <annotation directory> <frame number>”
